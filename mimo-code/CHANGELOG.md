# Changelog

## v3.2.0 (2026-07-06)

### ✨ 新增

- **mimo 0.1.5 升级** — 升级到最新版本，支持内置 Skills 体系
- **14 个内置 Skills** — 文档处理、学术写作、深度调研、自动科研等
- **Orchestrator 模式** — 委派式协调 agent，支持 git worktree 隔离
- **Cron + Loop 调度** — 5 字段 cron 表达式，持久化调度
- **视觉感知图片粘贴** — 非视觉模型自动选择 vision_model
- **Edit 精确匹配** — 默认精确字符串匹配

### 🔧 优化

- **Windows 原生安装器** — PowerShell 一行命令安装
- **$schema 自动迁移** — 从 opencode.ai 迁移到 mimo.xiaomi.com
- **重复检测器重写** — 消除对结构化内容的误杀

## v3.1.0 (2026-07-04)

### ✨ 新增

- **全新 Web UI** — 完全重写的用户界面，更现代的设计风格
- **完整 Markdown 渲染** — 支持标题、列表、表格、引用、代码块（带语言标识和一键复制）
- **工具调用可视化** — 显示 AI 执行的工具调用及其状态
- **打字指示器** — 三个跳动的点，更直观的等待反馈
- **消息操作** — 支持复制、重试功能
- **会话删除** — 悬停显示删除按钮，支持删除会话
- **消息计数** — 侧边栏显示每个会话的消息数量
- **4 种主题** — 深色、浅色、德古拉、北欧主题
- **骨架屏加载** — 加载时显示骨架屏而非空白
- **键盘快捷键** — Ctrl+N 新建会话、Ctrl+K 命令面板、Esc 关闭弹窗

### 🔧 优化

- **移动端适配** — 汉堡菜单、响应式布局、触摸友好
- **性能优化** — requestAnimationFrame 平滑滚动、事件委托
- **智能时间显示** — "刚刚"、"X 分钟前"等更人性化的时间格式
- **输入框自动调整** — 随内容自动伸缩高度
- **更精致的视觉设计** — 渐变品牌色、统一的圆角和阴影

### 🐛 修复

- **流式响应处理** — 优化 SSE/NDJSON 流式响应的解析
- **错误处理** — 更友好的错误提示和重试机制

## v3.0.0 (2026-07-02)

### ✨ 新增

- **Add-on 独立化** — MiMo Code 作为独立 Add-on 运行，自带侧边栏 Web UI
- **TCP 端口转发** — 解决 `mimo serve` 仅绑定 localhost 的问题，支持 bridge 网络模式
- **Ingress 侧边栏** — 通过 Supervisor ingress 在 HA 侧边栏内嵌 Web UI
- **Web UI 增强** — 模型选择、命令面板 Ctrl+K、主题切换、代理模式、提供商管理
- **Add-on 检测** — 组件自动检测 Add-on 通道，无需手动配置连接地址
- **架构重构** — 分离 Add-on（运行层）和 Custom Component（集成层）

## v2.1.0 (2026-07-02)

### ✨ 新增

- **HA Add-on 支持** — 新增 `mimo-code/` 目录，提供完整的 Home Assistant Add-on 支持
- **多架构构建** — 支持 `aarch64` / `amd64` / `armv7` 三种架构
- **s6-overlay 服务管理** — 使用 s6-overlay 管理 `mimo serve` 进程生命周期，崩溃自动重启
- **Supervisor 集成** — 集成可通过 Supervisor API 检测 add-on 运行状态
- **看门狗** — config.yaml 配置 `watchdog` 端点，Supervisor 自动检查健康状态

### 🔧 优化

- **多阶段 Docker 构建** — 第一阶段安装 npm 包，第二阶段仅复制原生二进制到 HA 基础镜像，无需 Node.js 运行时
- **配置驱动** — 通过 `bashio` 读取 add-on 配置，端口可自定义
