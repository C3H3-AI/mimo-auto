## 一、任务清单

### T1: P0-A 安全修复（server.py）

**问题**：文件写 API 无鉴权、CORS `*`、绑 0.0.0.0

**修复方案**：

#### 1.1 改绑 127.0.0.1

```python
# server.py:1750
# 改前
server = ThreadingMiMoServer(("0.0.0.0", PORT), MiMoProxyHandler)
# 改后
server = ThreadingMiMoServer(("127.0.0.1", PORT), MiMoProxyHandler)
```

**影响**：外部无法直连 8099，只能通过 HA ingress 访问。这正是 desired behavior。

#### 1.2 CORS 收窄

```python
# server.py end_headers()
# 改前
self.send_header("Access-Control-Allow-Origin", "*")
# 改后：只允许 ingress 来源
origin = self.headers.get("Origin", "")
if "172.30.32" in origin or "homeassistant" in origin:
    self.send_header("Access-Control-Allow-Origin", origin)
```

#### 1.3 文件写路径收窄

```python
# server.py _sanitize_fs_path()
# 改前
ALLOWED_PREFIXES = ["/data", "/config", "/usr/share/mimocode"]
# 改后：只允许 addon 自有目录
ALLOWED_PREFIXES = ["/data/mimocode", "/usr/share/mimocode"]
```

**删除** `/config` 和 `/usr/share` 的写权限（read 可保留）。

#### 1.4 GET 不允许写操作

```python
# server.py do_GET
# 移除这行
if self.path.startswith("/api/fs/write"):
    self._handle_fs_write()
```

#### 1.5 添加 IP 白名单（可选加固）

```python
INGRESS_ALLOW = {"172.30.32.2", "127.0.0.1"}

def _reject_if_not_ingress(self) -> bool:
    ip = self.client_address[0]
    if ip not in INGRESS_ALLOW:
        self.send_error(403, "Direct access denied; use HA ingress")
        return True
    return False
```

### T2: mimo serve Basic Auth

**问题**：mimo serve API 无鉴权

**修复方案**：

#### 2.1 环境变量注入

```yaml
# mimo-code/config.yaml
options:
  server_password: ""
schema:
  server_password: str?
```

```python
# server.py 启动时
password = os.environ.get("MIMOCODE_SERVER_PASSWORD", "")
if password:
    os.environ["MIMOCODE_SERVER_PASSWORD"] = password
```

#### 2.2 webui 复用 Basic Auth

```python
# server.py proxy to mimo serve
# 改前
async with session.post(f"{base_url}/session", json={}) as resp:

auth = None
password = os.environ.get("MIMOCODE_SERVER_PASSWORD", "")
if password:
    auth = aiohttp.BasicAuth("mimocode", password)
async with session.post(f"{base_url}/session", json={}, auth=auth) as resp:
```

### T6: 前端稳定性

#### 6.1 restore_webui.sh 加强

```bash
#!/bin/sh
SRC=/data/mimocode/webui
DST=/usr/share/mimocode/webui
if [ -d "${SRC}" ]; then
  # Python files
  for f in client.py channel_manager.py session_manager.py feishu_client.py tcp_proxy.py action_confirm.py ha_context.py evolution_review.py server.py; do
    [ -f "${SRC}/$f" ] && cp "${SRC}/$f" "${DST}/$f"
  done
  # Frontend dist
  [ -d "${SRC}/dist" ] && cp -r "${SRC}/dist/"* "${DST}/dist/" 2>/dev/null
  echo "[restore] WebUI files restored from /data"
fi
```

#### 6.2 确保 /data/dist 目录存在

每次部署后自动备份到 /data：
```bash
docker exec addon sh -c 'cp -r /usr/share/mimocode/webui/dist/ /data/mimocode/webui/dist/'
```
